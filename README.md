A high-performance, fully responsive personal portfolio designed to showcase my projects, technical skills, and development journey. Built with a focus on seamless user experience, interactive animations, and a serverless architecture.

🔗 **Live Demo:** https://portfolio-chi-ten-13.vercel.app/

---

## ✨ Features

* ⚡ Lightning-fast performance using Vite + React
* 🎨 Smooth animations with Framer Motion
* 📱 Fully responsive (mobile-first design)
* 📬 Serverless contact form using Web3Forms
* 🔒 Spam protection with hCaptcha

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Forms & Security:** Web3Forms API, hCaptcha
* **Deployment:** Vercel

---

## 🚀 Getting Started (Local Setup)

### 1. Clone the repository

git clone https://github.com/Dheeraj07-cmd/portfolio.git
cd portfolio

### 2. Install dependencies

npm install

### 3. Setup environment variables

Create a `.env` file and add:

VITE_WEB3FORMS_KEY=your_access_key_here

### 4. Run the project

npm run dev
